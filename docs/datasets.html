Dataset Explorer

Searchable and filterable inventory of IBD omics datasets

Author

Dahham Alsoud

This table lists all datasets currently indexed in the IBD Omics Explorer. Use the search box and filters to narrow results. All metadata are conservative annotations derived from public records; fields marked Unclear require manual verification. Dataset availability and access status should be confirmed before analysis.

Filters and Search

data = FileAttachment("data/ibd_omics_datasets.csv").csv({typed: true})
omicsOptions  = ["All", ...Array.from(new Set(
  data.flatMap(d => (d.omics_layer || "").split(";").map(s => s.trim()))
).values()).filter(Boolean).sort()]

diseaseOptions = ["All", "CD", "UC", "IBD", "Pediatric", "Multi-omics", "Controls"]

accessOptions = [
  "All",
  "Public raw data",
  "Public processed data",
  "Public portal",
  "Controlled access",
  "Publication only",
  "Unclear"
]

sampleOptions = ["All", ...Array.from(new Set(
  data.flatMap(d => (d.sample_type || "").split(";").map(s => s.trim()))
).values()).filter(Boolean).sort()]
viewof searchText = Inputs.text({
  label: "Search all fields",
  placeholder: "e.g. scRNA-seq, pediatric, colon, GWAS …",
  width: "100%"
})
viewof omicsFilter = Inputs.select(omicsOptions, {
  label: "Omics layer",
  value: "All"
})

viewof diseaseFilter = Inputs.select(diseaseOptions, {
  label: "Disease",
  value: "All"
})

viewof accessFilter = Inputs.select(accessOptions, {
  label: "Data access status",
  value: "All"
})

viewof sampleFilter = Inputs.select(sampleOptions, {
  label: "Sample type",
  value: "All"
})
filtered = {
  let result = data;

  if (omicsFilter !== "All") {
    result = result.filter(d =>
      (d.omics_layer || "").toLowerCase().includes(omicsFilter.toLowerCase())
    );
  }

  if (diseaseFilter !== "All") {
    result = result.filter(d =>
      (d.disease || "").toLowerCase().includes(diseaseFilter.toLowerCase())
    );
  }

  if (accessFilter !== "All") {
    result = result.filter(d =>
      (d.data_access_status || "").toLowerCase().includes(accessFilter.toLowerCase())
    );
  }

  if (sampleFilter !== "All") {
    result = result.filter(d =>
      (d.sample_type || "").toLowerCase().includes(sampleFilter.toLowerCase())
    );
  }

  if (searchText && searchText.trim() !== "") {
    const term = searchText.trim().toLowerCase();
    result = result.filter(d =>
      Object.values(d).some(v => v && String(v).toLowerCase().includes(term))
    );
  }

  return result;
}

resultCount = html`<p style="font-size:0.86rem;color:#5d5c57;margin:0.3rem 0 0.6rem 0;">
  Showing <strong>${filtered.length}</strong> of <strong>${data.length}</strong> datasets
</p>`

Data Access Status Legend

Label Meaning
Public raw data Raw data files (FASTQ, CEL, IDAT, mzML, count matrices, etc.) are publicly downloadable
Public processed data Processed expression/abundance matrix is available; raw data may not be public
Public portal Data are accessible via a public portal or database
Controlled access Indexed publicly but requires application or institutional approval
Publication only Described in a publication; reusable data files are not clearly available
Unclear Repository record exists but data availability or metadata are incomplete

Note. Data access status and metadata were assigned conservatively based on public records available at the time of curation. Availability may change. Always verify directly with the source repository before initiating analysis.
